## [0.3.0] - 2026-07-07

### 新增功能

#### 统计页面
- 新增统计页面 `/statistics`，作为一级页面（底部 TabBar 切换）
- 支持月/年两种统计维度切换
- 趋势图表：支持柱状图/折线图切换，展示收入、支出数据
- 分类统计环形图：按支出/收入分类展示占比，小于2%的分类不显示标签
- 明细表格：展示所有日期的数据（含无数据日期），支持收入、支出、结余列
- 统计卡片：展示筛选范围内的收入、支出、余额汇总

#### 报表统计功能
- 账本管理中新增"报表统计"入口，点击直接查看该账本统计数据
- 报表统计页面作为二级页面（隐藏底部 TabBar，显示返回按钮）
- 查看报表统计时不会切换当前账本，仅查看目标账本数据

### 优化改进

#### TabBar 导航优化
- 底部 TabBar 使用 `replace` 模式导航，避免历史记录堆积
- 在一级页面（记账/统计）按返回键直接退出应用

#### 图表交互优化
- 移除图表点击高亮边框和 Tooltip
- 环形图标签引线移除
- 环形图中间金额文字支持动态字号调整和超长换行展示

#### 金额显示优化
- 账单列表金额去掉强制换行（`whitespace-nowrap`）
- 统计页面明细表格金额去掉￥符号
- 新建账单金额输入去掉￥符号

#### 趋势图表优化
- 收入柱状图使用绿色，支出使用红色
- 移除"结余"选项，趋势图只显示收入和支出
- "全部"选项下同时显示收入和支出

### 修复问题

- 修复统计明细只显示有数据日期的问题，改为展示所有日期
- 修复分类统计百分比计算不准确问题
- 修复账单列表金额强制换行导致布局错乱问题

## [0.2.0] - 2026-07-04

### 新增功能

#### 多账本（钱包）管理
- 新增独立的账本管理页面 `/wallets`，支持创建、编辑、删除账本
- 账本支持自定义名称、描述与主题颜色（8 种预设色）
- 首页可通过左上角菜单进入账本管理，点击账本即可切换当前账本
- 首页账单列表、统计卡片、悬浮按钮均随当前账本主题色联动
- 支持账单迁移：将某账本的全部账单迁移至另一账本，操作不可逆并有二次确认
- 支持清空账本账单：一键清除指定账本下的所有账单（含二次确认）
- 默认账本不可删除，至少保留一个账本
- 账单数据新增 `walletId` 字段，按当前账本过滤展示
- 账本数据通过 localStorage 持久化（键名 `wallet-storage`）

#### 深色模式
- 新增深色模式主题切换，首页与分类管理页右上角提供切换按钮（月亮/太阳图标）
- 首次进入自动跟随系统偏好（`prefers-color-scheme`）
- 主题选择保存在 localStorage（键名 `theme`），下次进入自动应用
- 全量组件适配深色模式：账单列表、抽屉、弹窗、日期选择、统计卡片等

#### Toast 全局通知系统
- 新增全局 Toast 通知组件，支持 success / error / info / warning 四种类型
- 通知自动消失（默认 2500ms），支持手动关闭
- 提供 `useToast` Hook 封装快捷调用方法

#### 新增组件与 Hook
- `Loading` - 新增 `LoadingSpinner` 与 `LoadingOverlay` 加载组件
- `Empty` - 重写为可复用空状态组件，支持 default / bills / search 变体及自定义内容
- `Toast` - 新增 Toast 容器组件
- `useBillForm` - 新增账单表单可复用 Hook，封装状态管理、金额校验与提交逻辑
- `useToast` - 新增 Toast 便捷 Hook
- `useTheme` - 新增主题管理 Hook

#### 新增页面与路由
- 新增独立的新建/编辑账单页面 `/add/:id?`，替代原纯抽屉式交互
- 新增账本管理页面 `/wallets`

### 优化改进

#### 设计体系统一
- 统一设计令牌：新增 primary / expense / income 语义化颜色调色板，替代原 emerald / red / green
- 新增 card / modal / btn / input 圆角令牌与 card / modal / floating 语义化阴影
- 新增 slide-up / fade-in / scale-in / toast-in / toast-out 等动画与关键帧
- 新增语义化 CSS 组件类：card-base、btn-primary、btn-secondary、input-base、tab-container 等
- 全量组件迁移至语义化类名，统一视觉风格

#### 架构重构
- 抽取 `useBillForm` Hook 复用账单表单逻辑，消除 `AddBill` 与 `AddBillDrawer` 间重复代码
- 表单提交新增 `isSubmitting` 加载状态与 `canSubmit` 校验，保存按钮集成加载动画
- 优化 `Empty` 组件，支持多种空状态场景
- `StatCard` 与 `FloatingButton` 支持自定义主题色

#### 数据与存储
- `useBillStore` 新增 `clearBillsByWalletId` 与 `migrateBills` 方法
- 新增 `useWalletStore` 钱包状态管理（增删改查 + 切换 + localStorage 持久化）
- 新增 `useToastStore` 通知状态管理（消息队列 + 自动消失）
- `Bill` 类型新增可选 `walletId` 字段

### 修复问题

- 新建账单抽屉关闭时添加 `pointer-events-none`，避免误触
- 移除新增账单页金额输入框的宽度自适应测量逻辑，简化实现

### 技术实现

#### 新增核心组件
- `WalletManage` - 账本管理页
- `AddBill` - 独立新建/编辑账单页
- `Loading` - 加载组件
- `Toast` - 全局通知组件

#### 新增状态管理
- `useWalletStore` - 账本数据（增删改查 + 切换）
- `useToastStore` - Toast 通知（消息队列 + 自动消失）

#### 新增 Hook
- `useBillForm` - 账单表单逻辑复用
- `useToast` - Toast 通知便捷调用
- `useTheme` - 深色模式主题管理

### 已知限制

- 数据仅保存在当前浏览器 localStorage 中，无法跨设备同步
- 不支持数据导入/导出功能
- 不支持多币种
- 不支持账单搜索功能
- 统计仅支持按年月筛选，不支持自定义时间段

## [0.1.0] - 2026-07-03

### 新增功能

#### 首页账单列表
- 实现账单列表展示，支持按日期分组（同一天归为一组）
- 日期卡片显示当天收支汇总（支出红色/收入绿色）
- 顶部标题栏固定，点击日期区域弹出日期选择弹层
- 日期选择弹层支持按年/按月两种模式切换：
  - 按年模式：仅显示年份列表，筛选整年账单
  - 按月模式：显示年份 + 月份列表，筛选指定年月账单
- 日期选择弹层从顶部向下弹出，无标题栏，通过背景点击或底部按钮关闭
- 日期选择弹层打开时禁用背景页面滚动
- 年份自动跳转：无数据时自动切换到最近有数据的年份
- 统计卡片展示收入、支出、余额汇总
- 悬浮新增按钮（+），滚动时自动显隐
- 账单项左滑显示编辑/删除按钮
- 列表滚动时自动收起已展开的左滑操作

#### 新建/编辑账单抽屉
- 底部全屏抽屉式新建页面，URL 参数控制状态
- 支出/收入 Tab 切换，分类列表联动更新
- 分类网格选择（5列布局），默认选中第一个
- 分类选中高亮（支出红色、收入绿色），未选中白色底色
- 正方形分类按钮，图标大、留白小
- 备注输入框，铺满剩余空间
- 金额输入框，支持完整金额校验：
  - 禁止多个小数点
  - 禁止小数点开头（自动补0）
  - 多个0开头自动合并
  - 最多两位小数
  - 最小宽度父元素一半，最大宽度80%
- 日期选择弹窗，支持今天/昨天/前天快捷选择
- 日历组件支持月份切换和日期点选
- "再记"按钮支持连续新建
- "保存"按钮保存并返回
- 编辑功能：预填充数据，保存后关闭抽屉
- 返回按钮关闭抽屉，手机返回键适配

#### 账单操作
- 左滑手势：支持触摸滑动和鼠标拖拽
- 编辑功能：点击编辑按钮打开抽屉预填充数据
- 删除功能：二次确认弹窗，防止误删
- 保存后返回列表不滚动到顶部

#### 分类管理
- 独立分类管理页面 `/categories`
- 支出/收入 Tab 切换
- 分类网格展示（5列），与新建页风格一致
- 每个分类右上角删除按钮（减号图标）
- 删除保护：已有账单的分类禁止删除并提示
- 添加分类弹窗：名称输入 + 32个图标选择
- 拖拽排序：桌面端 HTML5 拖拽 + 移动端触摸拖拽
- 返回按钮回到新建抽屉

#### 数据与存储
- localStorage 本地持久化存储
- Zustand 状态管理
- 首次使用自动初始化默认分类和示例账单
- 默认8个支出分类、5个收入分类
- 7条示例账单数据

#### 交互体验
- 抽屉打开/关闭动画过渡
- 左滑操作动画反馈
- 按钮悬停和点击状态反馈
- 空状态友好提示
- 确认弹窗统一样式
- 响应式布局，移动端优先

### 技术实现

#### 技术栈
- React 18 + TypeScript
- Vite 构建工具
- Tailwind CSS 样式方案
- React Router v6 路由管理
- Zustand 状态管理
- Lucide React 图标库

#### 核心组件
- `Home` - 首页容器
- `AddBillDrawer` - 新建/编辑抽屉
- `CategoryManage` - 分类管理页
- `BillItem` - 账单列表项（支持左滑）
- `CategoryGrid` - 分类网格
- `StatCard` - 统计卡片
- `DatePicker` - 日期选择弹窗
- `ConfirmModal` - 确认弹窗
- `FloatingButton` - 悬浮按钮

#### 状态管理
- `useBillStore` - 账单数据（增删改查 + 统计）
- `useCategoryStore` - 分类数据（增删改 + 排序 + 使用检测）

### 修复问题

- 抽屉状态使用 URL 参数管理，修复浏览器返回按钮行为
- 修复从分类管理返回时抽屉抖动问题
- 修复 TypeScript 类型错误（lucide-react 图标导入）
- 优化左滑手势与页面滚动的冲突处理
- 优化金额输入框唤起键盘的灵敏度
- 修复日期选择弹层和新建账单抽屉打开时背景页面滚动的问题

### 已知限制

- 数据仅保存在当前浏览器 localStorage 中，无法跨设备同步
- 不支持数据导入/导出功能
- 不支持多币种
- 不支持账单搜索功能
- 统计仅支持按年月筛选，不支持自定义时间段
