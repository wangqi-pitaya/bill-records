## 1. 快速开始

### 1.1 环境要求

- Node.js >= 18
- npm 或 yarn

### 1.2 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 1.3 首次使用

首次打开应用时，系统会自动创建一个默认账本（"日常账本"）、一组默认分类（餐饮、交通、购物等）和几条示例账单，方便您快速了解功能。所有数据保存在浏览器本地，不会上传到服务器。

## 2. 首页 - 查看账单

### 2.1 界面布局

打开应用后进入首页，界面自上而下分为三个区域：

1. **标题栏**（固定在顶部）：左侧为账本管理入口（菜单图标），中间显示当前选中的年份或年月（点击弹出日期选择弹层），右侧为深色模式切换按钮
2. **统计卡片**：显示当前账本在筛选范围内的**收入**、**支出**和**余额**，卡片主题色随当前账本颜色变化
3. **账单列表**：仅显示当前账本下的账单，按日期倒序排列，同一天的数据归为一组

### 2.2 选择日期

点击标题栏的日期区域（如"2025年"或"2025年8月"），弹出日期选择弹层：

**按年模式**（默认）：
- 只显示年份列表，选择后查看整年的账单

**按月模式**：
- 显示年份列表 + 月份列表（1-12月）
- 选择年份后再选择具体月份，查看该月的账单

**操作方式**：
- 点击"按年"或"按月"切换模式
- 点击年份按钮选择年份
- 在按月模式下点击月份按钮选择月份
- 点击"确定"应用筛选，"取消"关闭弹层
- 点击弹层外部背景也可关闭

### 2.3 自动跳转

- 如果当前选择的年份没有账单数据，系统会自动跳转到有数据的最近年份
- 如果所有年份都没有数据，则停留在当前年份

### 2.4 查看账单详情

每条账单显示：
- 左侧：分类图标（圆形背景）
- 中间：分类名称（上方）+ 备注（下方）
- 右侧：金额（上方，收入绿色/支出红色）+ 日期（下方）

### 2.5 悬浮按钮

- 页面右下角有一个 `+` 悬浮按钮，点击打开新建账单页面
- 按钮颜色随当前账本主题色变化
- 向上滚动列表时按钮自动隐藏，向下滚动时显示

## 3. 新建账单

### 3.1 打开新建页面

点击首页右下角的悬浮按钮 `+`，进入新建账单页面。

### 3.2 选择收支类型

页面顶部中间有"支出"和"收入"两个 Tab：
- **支出**（红色）：记录消费支出
- **收入**（绿色）：记录收入来源

点击切换类型，下方分类列表会自动更新。

### 3.3 选择分类

- 分类以网格形式展示，默认选中第一个
- 点击分类即可选中，选中的分类会高亮显示
- 分类图标下方显示分类名称

### 3.4 填写备注

在分类下方的输入框中填写账单备注（可选），例如"午餐"、"打车"等。

### 3.5 输入金额

- 点击金额输入框，输入数字
- 系统会自动校验金额格式：
  - 禁止输入多个小数点
  - 禁止以小数点开头（自动补0）
  - 多个0开头自动合并
  - 最多保留两位小数
- 输入框左侧显示 ¥ 符号

### 3.6 选择日期

- 点击"今天"按钮，弹出日历弹窗
- 弹窗顶部有快捷选项：**今天**、**昨天**、**前天**
- 也可以切换月份后点击具体日期
- 点击"确定"确认选择，"取消"关闭弹窗

### 3.7 保存账单

**保存**：点击右下角"保存"按钮，账单保存并返回首页。保存时按钮会显示加载动画，期间不可重复点击。

**再记（连续记账）**：点击左下角"再记"按钮，保存当前账单并清空金额和备注，继续记下一笔。

> 注意：必须选择了分类并输入了金额，保存/再记按钮才可用。新建的账单会自动归属到当前选中的账本。

## 4. 编辑账单

### 4.1 打开编辑

在首页账单列表中，**向左滑动**某条账单，露出右侧的编辑和删除按钮，点击**编辑按钮**（铅笔图标），进入编辑账单页面。

### 4.2 修改内容

页面打开后会预填充当前账单的所有数据：
- 收支类型
- 分类（自动选中）
- 金额
- 备注
- 日期

修改需要更新的字段即可。

### 4.3 保存编辑

点击"保存"按钮，更新账单数据并返回首页。

---

## 5. 删除账单

### 5.1 触发删除

向左滑动账单，点击**删除按钮**（垃圾桶图标）。

### 5.2 确认删除

系统会弹出确认对话框，显示：
> "确定要删除这条账单记录吗？删除后无法恢复。"

- 点击"删除"确认删除
- 点击"取消"取消操作

## 6. 分类管理

### 6.1 进入管理页

在新建/编辑账单页面中，点击右上角的**设置图标**（齿轮），进入分类管理页面。页面右上角还提供深色模式切换按钮。

### 6.2 切换收支分类

页面顶部同样有"支出"和"收入"Tab，点击切换查看不同类型的分类。

### 6.3 添加分类

1. 点击网格底部的"添加"按钮（+ 图标）
2. 在弹窗中输入分类名称
3. 从图标库中选择一个图标（32个图标可选）
4. 点击"添加"确认，"取消"关闭弹窗

### 6.4 删除分类

- 每个分类右上角有一个减号图标按钮
- 点击后如果该分类**没有**被任何账单使用，直接删除
- 如果该分类**已有账单记录**，会提示"该分类已有账单记录，无法删除"

> ⚠️ **注意**：删除分类前请确保没有账单使用该分类，否则只能先删除相关账单。

### 6.5 排序分类

支持拖拽调整分类顺序：

**桌面端**：
- 鼠标按住分类卡片，拖动到目标位置后松开

**移动端**：
- 长按分类卡片约 500ms 后拖动
- 拖到目标位置后松开手指

排序结果会自动保存。

### 6.6 返回

点击左上角的返回箭头，返回上一页。

## 7. 账本管理

### 7.1 进入账本管理

点击首页左上角的**菜单图标**，进入"我的账本"管理页面。

### 7.2 切换账本

- 账本以卡片列表形式展示，每张卡片显示账本名称、描述及账单数量
- 点击任意账本卡片即可切换为当前账本，并自动返回首页
- 首页的账单列表、统计卡片、悬浮按钮会随当前账本主题色联动

### 7.3 添加账本

1. 点击页面右上角的 `+` 按钮，弹出"添加账本"弹窗
2. 输入账本名称（必填，最多 20 字）和描述（可选，最多 50 字）
3. 从 8 种预设主题色中选择一个
4. 点击"添加"确认创建

### 7.4 编辑账本

1. 点击账本卡片右侧的**设置图标**，打开底部设置弹层
2. 点击"修改"，进入编辑弹窗
3. 修改名称、描述或主题色后点击"保存"

### 7.5 迁移账单

- 在账本设置弹层中点击"迁移账本"，可将当前账本的全部账单迁移至其他账本
- 在迁移弹窗中选择目标账本后点击"迁移"
- 迁移操作不可逆，请谨慎操作
- 迁移完成后会弹出成功通知

### 7.6 清除账单

- 在账本设置弹层中点击"清除账单"，可一键清空该账本下的所有账单
- 系统会弹出二次确认弹窗，点击"清除"确认操作
- 此操作不可恢复，但不会删除账本本身

### 7.7 删除账本

- 在账本设置弹层中点击"删除账本"（默认账本不显示此选项）
- 系统会弹出二次确认弹窗，点击"删除"确认
- 删除账本时，该账本下的账单数据也会一并删除
- 默认账本不可删除，且至少需要保留一个账本

### 7.8 报表统计

- 在账本设置弹层中点击"报表统计"，查看该账本的统计数据
- 此操作不会切换当前账本，仅查看目标账本数据
- 报表统计页面作为二级页面，显示返回按钮，点击返回账本管理页

## 8. 统计页面

### 8.1 进入统计页面

**方式一**：点击底部 TabBar 的"统计"图标，进入当前账本的统计页面（一级页面）

**方式二**：在账本管理页面，点击某账本的设置图标 → "报表统计"，进入该账本的统计页面（二级页面）

### 8.2 页面布局

统计页面自上而下分为四个区域：

1. **标题栏**：
   - 一级页面：显示"月"/"年"切换按钮，用于选择统计维度
   - 二级页面：显示返回按钮和"统计"标题

2. **统计卡片**：显示筛选范围内的**收入**、**支出**和**余额**汇总

3. **趋势图表**：展示收入、支出趋势数据，支持柱状图和折线图切换

4. **分类统计**：环形图展示各分类占比

5. **明细表格**：按日/月展示详细收支数据

### 8.3 切换统计维度

点击标题栏的"月"或"年"按钮切换统计维度：

- **月统计**：按日统计，展示当月每天的收支情况
- **年统计**：按月统计，展示当年每月的收支情况

### 8.4 选择日期

点击标题栏的日期区域（如"2025年8月"），弹出日期选择弹层，选择需要统计的月份或年份。

### 8.5 趋势图表

#### 8.5.1 图表类型切换

- **柱状图**：以柱状形式展示每日/每月的收入和支出
- **折线图**：以折线形式展示收入和支出趋势变化

#### 8.5.2 数据筛选

图表下方有三个选项按钮：

- **支出**：只显示支出数据
- **收入**：只显示收入数据
- **全部**：同时显示收入和支出

#### 8.5.3 颜色说明

- 收入：绿色
- 支出：红色

### 8.6 分类统计

#### 8.6.1 切换收支类型

点击"支出"或"收入"按钮，切换查看不同类型的分类占比。

#### 8.6.2 环形图说明

- 环形图展示各分类的金额占比
- 占比小于2%的分类不显示标签，避免标签拥挤
- 环形图中间显示该类型的总金额

### 8.7 明细表格

#### 8.7.1 表格内容

- **日期**：显示具体日期（月统计显示日，年统计显示月）
- **收入**：当日/当月收入金额
- **支出**：当日/当月支出金额
- **结余**：当日/当月结余（收入 - 支出）

#### 8.7.2 数据范围

表格展示所选时间范围内的**所有日期**，包括无收支数据的日期（显示为0）。

#### 8.7.3 平均值

表格上方显示日均/月均支出金额。

## 9. 深色模式

### 9.1 切换主题

- 在首页或分类管理页右上角点击**月亮/太阳图标**即可切换深色/浅色模式
- 首次打开应用时，系统会自动跟随操作系统的深色模式偏好

### 9.2 主题记忆

- 主题选择会自动保存在浏览器本地（键名 `theme`）
- 下次打开应用时自动应用上次选择的主题

### 9.3 适配范围

深色模式已适配全部界面，包括：账单列表、新建/编辑账单页、分类管理页、账本管理页、统计页面、日期选择弹窗、确认弹窗、Toast 通知等。

## 10. 数据存储说明

### 10.1 存储位置

所有数据存储在浏览器的 **localStorage** 中，包含以下键：

| 键名 | 内容 | 示例 |
|-----|------|------|
| `bill_records` | 账单数据 | `[{id, type, category, walletId, ...}]` |
| `categories` | 分类数据 | `[{id, name, icon, type}]` |
| `wallet-storage` | 账本数据 | `{wallets: [...], currentWalletId: "..."}` |
| `theme` | 主题偏好 | `"light"` 或 `"dark"` |

### 10.2 数据安全

- 数据仅保存在当前浏览器中
- 清除浏览器数据会导致记录丢失
- 不同浏览器/设备之间数据不互通
- 建议定期导出重要数据（后续版本支持）

### 10.3 重置数据

如需重置为初始状态，可在浏览器控制台执行：

```javascript
localStorage.removeItem('bill_records');
localStorage.removeItem('categories');
localStorage.removeItem('wallet-storage');
localStorage.removeItem('theme');
location.reload();
```

## 11. 常见问题

**Q: 为什么切换年份时有时会自动跳转？**  
A: 如果当前选择的年份没有账单数据，系统会自动跳转到有数据的最近年份，避免显示空列表。

**Q: 左滑账单时列表也跟着滚动怎么办？**  
A: 当列表正在滚动时，左滑操作会被禁用，等滚动停止后再尝试。

**Q: 分类删除按钮是灰色的点不了？**  
A: 该分类可能已有账单在使用，需要先删除或修改相关账单后才能删除分类。

**Q: 新建的账单为什么在首页看不到？**  
A: 账单会归属到当前选中的账本。请在首页左上角菜单中确认当前账本是否正确，切换到对应账本即可查看。

**Q: 默认账本可以删除吗？**  
A: 不可以。默认账本（"日常账本"）不可删除，且系统至少需要保留一个账本。

**Q: 迁移账单后原账本的账单还在吗？**  
A: 迁移操作会将原账本的全部账单转移至目标账本，原账本将不再保留这些账单。该操作不可逆，请谨慎操作。

**Q: 手机返回键直接退出应用了？**  
A: 当弹窗打开时，按返回键会先关闭弹窗。如果弹窗已关闭，再次按返回键才会退出。

**Q: 数据会同步到云端吗？**  
A: 当前版本（v0.2.0）所有数据仅保存在本地浏览器中，不会上传到任何服务器。
