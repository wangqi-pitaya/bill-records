## 1. 快速开始

### 1.1 环境要求

- Node.js >= 18
- npm 或 yarn

### 1.2 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 1.3 首次使用

首次打开应用时，系统会自动创建一组默认分类（餐饮、交通、购物等）和几条示例账单，方便您快速了解功能。所有数据保存在浏览器本地，不会上传到服务器。

## 2. 首页 - 查看账单

### 2.1 界面布局

打开应用后进入首页，界面自上而下分为三个区域：

1. **标题栏**（固定在顶部）：显示当前选中的年份或年月，点击弹出日期选择弹层
2. **统计卡片**：显示当前筛选范围内的**收入**、**支出**和**余额**
3. **账单列表**：按日期倒序排列，同一天的数据归为一组

### 2.2 选择日期

点击标题栏的日期区域（如"2025年"或"2025年8月"），弹出日期选择弹层：

**按年模式**（默认）：
- 只显示年份列表，选择后查看整年的账单

**按月模式**：
- 显示年份列表 + 月份列表（1-12月）
- 选择年份后再选择具体月份，查看该月的账单

**操作方式**：
- 点击"按年"或"按月"切换模式
- 点击年份按钮选择年份
- 在按月模式下点击月份按钮选择月份
- 点击"确定"应用筛选，"取消"关闭弹层
- 点击弹层外部背景也可关闭

### 2.3 自动跳转

- 如果当前选择的年份没有账单数据，系统会自动跳转到有数据的最近年份
- 如果所有年份都没有数据，则停留在当前年份

### 2.4 查看账单详情

每条账单显示：
- 左侧：分类图标（圆形背景）
- 中间：分类名称（上方）+ 备注（下方）
- 右侧：金额（上方，收入绿色/支出红色）+ 日期（下方）

### 2.5 悬浮按钮

- 页面右下角有一个绿色的 `+` 悬浮按钮，点击打开新建抽屉
- 向上滚动列表时按钮自动隐藏，向下滚动时显示

## 3. 新建账单

### 3.1 打开新建抽屉

点击首页右下角的悬浮按钮 `+`，从底部弹出全屏抽屉。

### 3.2 选择收支类型

抽屉顶部中间有"支出"和"收入"两个 Tab：
- **支出**（红色）：记录消费支出
- **收入**（绿色）：记录收入来源

点击切换类型，下方分类列表会自动更新。

### 3.3 选择分类

- 分类以网格形式展示，默认选中第一个
- 点击分类即可选中，选中的分类会高亮显示
- 分类图标下方显示分类名称

### 3.4 填写备注

在分类下方的输入框中填写账单备注（可选），例如"午餐"、"打车"等。

### 3.5 输入金额

- 点击金额输入框，输入数字
- 系统会自动校验金额格式：
  - 禁止输入多个小数点
  - 禁止以小数点开头（自动补0）
  - 多个0开头自动合并
  - 最多保留两位小数
- 输入框左侧显示 ¥ 符号

### 3.6 选择日期

- 点击"今天"按钮，弹出日历弹窗
- 弹窗顶部有快捷选项：**今天**、**昨天**、**前天**
- 也可以切换月份后点击具体日期
- 点击"确定"确认选择，"取消"关闭弹窗

### 3.7 保存账单

**保存**：点击右下角"保存"按钮，账单保存并关闭抽屉返回首页。

**再记（连续记账）**：点击左下角"再记"按钮，保存当前账单并清空金额和备注，继续记下一笔。

> 注意：必须选择了分类并输入了金额，保存/再记按钮才可用。

## 4. 编辑账单

### 4.1 打开编辑

在首页账单列表中，**向左滑动**某条账单，露出右侧的编辑和删除按钮，点击**编辑按钮**（铅笔图标）。

### 4.2 修改内容

抽屉打开后会预填充当前账单的所有数据：
- 收支类型
- 分类（自动选中）
- 金额
- 备注
- 日期

修改需要更新的字段即可。

### 4.3 保存编辑

点击"保存"按钮，更新账单数据并关闭抽屉。

---

## 5. 删除账单

### 5.1 触发删除

向左滑动账单，点击**删除按钮**（垃圾桶图标）。

### 5.2 确认删除

系统会弹出确认对话框，显示：
> "确定要删除这条账单记录吗？删除后无法恢复。"

- 点击"删除"确认删除
- 点击"取消"取消操作

## 6. 分类管理

### 6.1 进入管理页

在新建/编辑抽屉中，点击右上角的**设置图标**（齿轮），进入分类管理页面。

### 6.2 切换收支分类

页面顶部同样有"支出"和"收入"Tab，点击切换查看不同类型的分类。

### 6.3 添加分类

1. 点击网格底部的"添加"按钮（+ 图标）
2. 在弹窗中输入分类名称
3. 从图标库中选择一个图标（32个图标可选）
4. 点击"添加"确认，"取消"关闭弹窗

### 6.4 删除分类

- 每个分类右上角有一个减号图标按钮
- 点击后如果该分类**没有**被任何账单使用，直接删除
- 如果该分类**已有账单记录**，会提示"该分类已有账单记录，无法删除"

> ⚠️ **注意**：删除分类前请确保没有账单使用该分类，否则只能先删除相关账单。

### 6.5 排序分类

支持拖拽调整分类顺序：

**桌面端**：
- 鼠标按住分类卡片，拖动到目标位置后松开

**移动端**：
- 长按分类卡片约 500ms 后拖动
- 拖到目标位置后松开手指

排序结果会自动保存。

### 6.6 返回

点击左上角的返回箭头，返回新建抽屉页面。

## 7. 数据存储说明

### 7.1 存储位置

所有数据存储在浏览器的 **localStorage** 中，包含两个键：

| 键名 | 内容 | 示例 |
|-----|------|------|
| `bill_records` | 账单数据 | `[{id, type, category, ...}]` |
| `categories` | 分类数据 | `[{id, name, icon, type}]` |

### 7.2 数据安全

- 数据仅保存在当前浏览器中
- 清除浏览器数据会导致记录丢失
- 不同浏览器/设备之间数据不互通
- 建议定期导出重要数据（后续版本支持）

### 7.3 重置数据

如需重置为初始状态，可在浏览器控制台执行：

```javascript
localStorage.removeItem('bill_records');
localStorage.removeItem('categories');
location.reload();
```

## 8. 常见问题

**Q: 为什么切换年份时有时会自动跳转？**  
A: 如果当前选择的年份没有账单数据，系统会自动跳转到有数据的最近年份，避免显示空列表。

**Q: 左滑账单时列表也跟着滚动怎么办？**  
A: 当列表正在滚动时，左滑操作会被禁用，等滚动停止后再尝试。

**Q: 分类删除按钮是灰色的点不了？**  
A: 该分类可能已有账单在使用，需要先删除或修改相关账单后才能删除分类。

**Q: 手机返回键直接退出应用了？**  
A: 当抽屉或弹窗打开时，按返回键会先关闭抽屉/弹窗。如果抽屉已关闭，再次按返回键才会退出。

**Q: 数据会同步到云端吗？**  
A: 当前版本（v0.1.0）所有数据仅保存在本地浏览器中，不会上传到任何服务器。
